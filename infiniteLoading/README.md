# Infinite Loading

Loads indefinitely with the classic dot dot dot animation.  
Changes color on every animation reset, never the same color twice in a row.

    Chargement.  
    # screen clears
    
    Chargement..  
    # screen clears
    
    Chargement...  
    # screen clears
    # Changes color and repeats
